# Invitation app - Broccoli & Co.

Concepts and features used:
- MVVM architecture
- Retrofit library
- Coroutines for network interactions
- AndroidViewModel, LiveData and ViewBinding
- SharedPreferences to save the user's state
- AlertDialogs
